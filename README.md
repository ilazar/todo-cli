# todo cli

Todo CLI app running on Node.js.

Todo
  - id: string
  - text: string
  - completed: boolean
  - updated: date
  - version: number

CLI
  - show ['all'|'active'|'completed']
  - add 'todo-text'
  - update 'todo-id' 'todo-text'
  - done 'todo-id'
  - delete 'todo-id'
  - clear ['all'|'active'|'completed']
